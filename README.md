## 🚀 About This Project

This interactive portfolio was built using cutting-edge web technologies:

- **React Three Fiber** - For creating 3D graphics in React
- **Three.js** - Powerful 3D library
- **Drei** - Helpful utilities for React Three Fiber
- **GSAP** - Animation library for smooth camera movements
- **GLSL Shaders** - Custom shader effects
- **Blender** - 3D modeling software

## 🎯 Key Features

- Immersive 3D environment with realistic textures
- Scroll-controlled camera navigation
- Animated text elements that appear during the tour
- Custom shader effects for dynamic visuals
- Responsive design for all devices
